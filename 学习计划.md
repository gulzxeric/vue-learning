# Vue 学习计划

> **定位**：已掌握 HTML/CSS/JS/Node.js，零 Vue 基础
> **目标**：能独立做项目、应对面试
> **时间**：每天 3-4 小时，预计 4-6 周
> **技术栈**：Vue 3 + Composition API + `<script setup>` + Vite

---

## 第 1 周：Vue 核心基础（进度加快）

### Day 1-2：声明式渲染

**学什么：**
- Vue 是什么、SPA 概念
- 用 Vite 创建 Vue 项目
- `{{ }}` 插值表达式、`v-html`
- `v-bind` 动态属性绑定

**实践：** 做一个动态计数器

### Day 3-4：条件渲染 + 列表渲染

**学什么：**
- `v-if` / `v-else` / `v-show`（区别和性能考虑）
- `v-for` 循环渲染，key 的重要性
- 数组响应式方法

**实践：** 待办事项列表（增删功能）

### Day 5-6：事件处理 + 表单绑定

**学什么：**
- `v-on`（`@click` 等事件缩写）
- `v-model` 双向绑定原理
- `v-model` 修饰符（.lazy, .number, .trim）
- 事件修饰符（.stop, .prevent）

**实践：** 注册表单 + 简单表单验证

### Day 7：复习 + 小项目

**项目：** 个人信息收集表
- 包含条件显示（如选"学生"才显示学号输入）
- 列表展示（如已添加的标签）
- 表单绑定

---

## 第 2 周：组件化开发

### Day 8-9：组件基础

**学什么：**
- 组件定义、注册
- `<template>` 结构
- `props` 父传子（单向数据流）
- `emit` 子传父
- 组件间通信方式概览

**实践：** 把待办事项拆成独立组件

### Day 10-11：组件高级

**学什么：**
- `<script setup>` 语法糖
- `defineProps` / `defineEmits`
- 插槽（slot）：默认、具名、作用域插槽
- 组件 ref 访问子组件

**实践：** 可复用卡片组件 + 列表组件

### Day 12-13：组合式 API（Composition API）— 重点

**学什么：**
- `ref` vs `reactive` 的区别和使用场景
- `computed` 计算属性
- `watch` / `watchEffect` 侦听器
- 生命周期钩子（`onMounted`, `onUnmounted` 等）

**实践：** 用组合式 API 重写之前的项目

### Day 14：复习 + 小项目

**项目：** 商品展示页面
- 搜索过滤功能
- 分类切换
- 卡片组件复用

---

## 第 3 周：路由 + 状态管理 + API

### Day 15-17：Vue Router

**学什么：**
- 安装配置、路由表定义
- `<router-link>` / `<router-view>`
- 动态路由（`:id`）
- 嵌套路由
- 路由守卫（导航守卫）
- 编程式导航（`router.push`）
- 路由懒加载

**实践：** 多页面 SPA（首页 + 文章详情 + 关于页）

### Day 18-20：Pinia 状态管理 — 重点

**学什么：**
- 为什么需要状态管理（组件通信的局限）
- Pinia 基本用法：store、state、getter、action
- setup store 写法
- 多 store 管理
- Store 持久化

**实践：** 购物车项目 + 全局状态管理

### Day 21：API 请求

**学什么：**
- Axios 基础使用
- async/await 配合 Vue
- 数据加载状态 + 错误处理
- 封装统一请求模块

**实践：** 调用公开 API（如 JSONPlaceholder）做数据展示

---

## 第 4 周：TypeScript + 综合项目

### Day 22-25：综合项目（选一个）

**项目选择：**

| 项目 | 涉及技能 |
|------|----------|
| 电商商城 | 商品列表、搜索、购物车、详情页、分类 |
| 博客系统 | 文章列表、详情、评论、分类标签 |
| 高级 Todo | 分类、优先级、搜索、本地持久化 |

**涉及全部技能：** 组件化、路由、状态管理、API、TypeScript

### Day 26：TypeScript 基础（针对 Vue）

**重点学：**
- 基本类型：`string`, `number`, `boolean`, `array`, `object`
- 接口（interface）：定义 props、API 数据结构
- 类型推导和类型断言
- Vue + TS：`<script setup lang="ts">`
- `defineProps<T>()`, `defineEmits<T>()`
- `Ref<T>`, `ComputedRef<T>` 理解

**不需要学：** 高级泛型、装饰器、命名空间等冷门内容

### Day 27：项目优化

- 路由懒加载
- `v-memo` 等性能优化
- 组件抽离和代码组织
- 错误边界处理

## 第 5 周（可选）：进阶 + 面试冲刺

### 常见面试题

| 类别 | 考点 |
|------|------|
| 响应式 | `Object.defineProperty` vs `Proxy`、响应式原理 |
| 对比题 | `v-if` vs `v-show`、`computed` vs `watch`、`ref` vs `reactive` |
| 生命周期 | 8 个钩子函数及触发时机 |
| 组件通信 | props/emit、provide/inject、Pinia、event bus |
| 虚拟 DOM | Diff 算法思路、key 的作用 |
| Router | 路由守卫流程、hash vs history 模式 |
| Keep-alive | 作用、`activated`/`deactivated` 钩子 |
| 性能优化 | 懒加载、v-memo、虚拟列表、按需加载 |

### 进阶方向（按兴趣选学）
- Vue SSR（Nuxt.js）
- Vue 插件开发
- 单元测试（Vitest）
- 大项目架构设计

## 关键提醒

1. **一定用 Vue 3 + Composition API + `<script setup>`**，这是目前主流和面试标准
2. **用 Vite 创建项目**，不要用旧的 Vue CLI
3. **手写代码，不要复制粘贴**——这是学会编程的唯一方式
4. **遇到不懂的先查官方文档**，养成看文档的习惯
5. **每个练习都要自己敲一遍**，看懂不等于会写
